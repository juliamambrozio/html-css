Para funcionar a opção de dowload, é só acrescentar a tag a, download="nome do arquivo.formato e type="application/formato. (conferir o formato do application no site: iana.org/assignments/media-types)"

Para fazer o download se caso não funcioonar clicando em cima, é só clicar com o botão direito e salvar link
